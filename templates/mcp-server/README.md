# MCP Server Template

A Model Context Protocol (MCP) server for the NEAR ecosystem.

## Setup

```bash
npm install
npm run build
```

## Usage

```bash
# Run the server
npm start

# Development mode
npm run dev
```

## Tools

| Tool | Description |
|------|-------------|
| `example_tool` | An example tool — replace with your implementation |

## Development

```bash
npm test        # Run tests
npm run lint    # Lint source
npm run build   # Compile TypeScript
```
